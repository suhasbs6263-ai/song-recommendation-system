# 🎵 AI Song Recommendation System

## 📌 Project Overview

The **AI Song Recommendation System** is a web-based application that recommends songs based on user selection.
It uses a dataset containing song information and YouTube links to suggest similar songs.

Users can:

* Select a song
* Get recommended songs
* Play songs directly on YouTube
* Add songs to a playlist
* Explore an interactive music dashboard

The system is built using **Python, Flask, Machine Learning, HTML, CSS, and JavaScript**.

# ⚙️ Installation Guide

Follow the steps below to run the project on your system.

---

# 1️⃣ Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/song-recommendation-system.git
```

Navigate to the project folder:

```bash
cd song-recommendation-system
```

---

# 2️⃣ Create a Virtual Environment

```bash
python -m venv env
```

This creates a virtual environment folder named **env**.

---

# 3️⃣ Activate the Virtual Environment

## 🪟 Windows

```bash
env\Scripts\activate
```

---

## 🍎 Mac / Linux

```bash
source env/bin/activate
```

After activation you will see:

```
(env)
```

in the terminal.

---

# 4️⃣ Install Required Libraries

Install all dependencies using:

```bash
pip install -r requirements.txt
```

If you do not have a requirements file, install manually:

```bash
pip install flask pandas numpy scikit-learn
```

---

# ▶️ Run the Application

Start the Flask server:

```bash
python app.py
```

The application will start on:

```
http://127.0.0.1:5000
```

Open this link in your browser.
